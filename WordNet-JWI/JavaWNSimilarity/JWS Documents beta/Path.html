Path - beta

David Hope, 2008, University Of Sussex



Use:


path.path( cat, 1, dog, 2, n )

path.path( cat, dog, 2, n )

path.path( cat, 1, dog, n )

path.path( cat, dog, n )

path.max( cat, dog, n )



Brief Description:


Returns the reciprocal of the length of the shortest (geodesic) path between 2 synsets e.g. a shortest path between [a] and [v] might be: [a] ... [q] ... [t] ... [v] = a path length of 4 = 1/4 = 0.25


Note that we count the 'nodes' (synsets) in the path, not the links (as per Perl version)


POS Types Allowed:


nouns, verbs (as per Perl version)


Comments:


Fast but dumb method of getting similarity as we are simply applying a notion of 'semantic relatedness' via node counting.
