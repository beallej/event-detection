LeacockAndChodorow - beta

David Hope, 2008, University Of Sussex


Applies the basic (uses 'fake roots') version of the similarity measure as described in Leacock & Chodorow, 1998.


( word1, word1sense#, word2, word2sense#, POS )


Use:


lch.lch( cat, 1, dog, 2, n )

lch.lch( cat, dog, 2, n )

lch.lch( cat, 1, dog, n )

lch.lch( cat, dog, n )

lch.max( cat, dog, n )



Brief Description:


lch = -log(length / 2 X D)


where:

length = geodesic path (sense X, sense Y)

D = max. depth of <noun|verb> taxonomy
