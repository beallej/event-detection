JWSRandom – beta.02

David Hope, 2008, University Of Sussex




Generates random numbers which act as scores for sense pairs.


Implementation as per the Perl version .


Use as baseline



Use:


3 Constructors:


1. Default

Random random = new Random(dict);


2. Store scores for specific sense pairs


Random random = new Random(dict, true);


3. Set the upper limit on the scores


Random random = new Random(dict, true, 16.0);


random.random( cat, 1, dog, 2, n )

random.random( cat, dog, 2, n )

random.random( cat, 1, dog, n )

random.random( cat, dog, n )

random.max( cat, dog, n )
