AdaptedLeskTanimoto - beta

David Hope, 2008, University Of Sussex


Gets the similarity between any combination of


( word1, word1sense#, word2, word2sense#, POS )


Use:


lesk.lesk( cat, 1, dog, 2, n )

lesk.lesk( cat, dog, 2, n )

lesk.lesk( cat, 1, dog, n )

lesk.lesk( cat, dog, n )

lesk.max( cat, dog, n )



Brief Description:


There are many ways in which one could adapt the original Lesk measure to WordNet – there is no right way in which to do this: this is just one way in which we could apply Lesk's overlap measure to WordNet.


Here, we get the glosses of all synsets that have a direct WordNet relation with each of our input senses. We also get the hyponyms of a hypernym of a sense. We consider all the words in these glosses as a 'supergloss'. Supergloss words must not be on the stop list (stoplist.out), they are also lemmatised. We then construct two vectors: one for each supergloss. Each vector contains the count of each word found in the supergloss, and , rather obviously, the basis set for the vectors is the combined words in both superglosses. We then proceed to calculate the Jaccard-Tanimoto Coefficient: this is our 'Lesk' score.



Issues:

Can be slow, as we get all the (immediate) hyponyms of a hypernym, which, as you can imagine, could be quite a large set for certain words. There's no hack around this.


Refs:


Banerjee S. and Pedersen T. 2002. An Adapted Lesk Algorithm for Word Sense Disambiguation Using WordNet. In Proceeding of the Fourth International Conference on Computational Linguistics and Intelligent Text Processing (CICLING-02). Mexico City.
