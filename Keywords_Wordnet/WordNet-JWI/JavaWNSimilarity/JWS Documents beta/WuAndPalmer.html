WuAndPalmer - beta

David Hope, 2008, University Of Sussex


Applies the Perl version* of the Wu & Palmer (Wu & Palmer, 1994) measure to get the similarity between any combination of


( word1, word1sense#, word2, word2sense#, POS )


Use:


wp.wp( cat, 1, dog, 2, n )

wp.wp( cat, dog, 2, n )

wp.wp( cat, 1, dog, n )

wp.wp( cat, dog, n )

wp.max( cat, dog, n )



Brief Description:


wp = ( 2 X depth(lcs) ) / ( depth(synset1) + depth(synset2) )


where:

lcs =the least common subsumer of synset1 and synset2




Refs:

Wu & Palmer (1994)


* The Perl version counts nodes not edges
