Lin - beta

David Hope, 2008, University Of Sussex


Applies Lin's measure to get the similarity between any combination of


( word1, word1sense#, word2, word2sense#, POS )


Use:


lin.lin( cat, 1, dog, 2, n )

lin.lin( cat, dog, 2, n )

lin.lin( cat, 1, dog, n )

lin.lin( cat, dog, n )

lin.max( cat, dog, n )



Brief Description:


lin = 2 * IC(lcs) / (IC(synset1) + IC(synset2))


where:

lcs =the least common subsumer of synset1 and synset2

IC = the Information Content (of a synset).



Refs:


Lin D. 1998. An information-theoretic definition of similarity. In Proceedings of the 15th International Conference on Machine Learning. Madison, WI.
